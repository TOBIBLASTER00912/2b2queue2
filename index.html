About the Bot

2b2queue provides real-time statistics and updates for the Minecraft server 2b2t. Whether you're looking for queue statistics, in-game player counts, or the latest MOTD, this bot has you covered. Stay informed and never miss an update with our feature-rich and user-friendly bot.

Discord Bot Command list

  • /queue - Displays the current queue data
  • /uptime - Displays the current uptime of the Website/API/Bot
  • /ingame - Displays the count of the current online players on 2b2t
  • /motd - Displays the current Message of the Day (motd)
  • /api - Gives the current working API link
  • (There are a few more, but they are staff only. Command list can also be accessed from top.gg with the Button below.)
